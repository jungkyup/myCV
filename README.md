MyCV
=====================================

## About

This is my Curriculum Vitae, made using `XeLaTeX`. 

## Installation

To install `XeLaTeX`, download one of the distributions from http://www.tug.org/mactex/

In order to compile this, I run

    $ xelatex cv.tex
